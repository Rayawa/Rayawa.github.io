关于我

从通信工程出发，在软件、生物、影像与创作实践中建立自己的长期路线。

个人简介

我目前就读于北方工业大学通信工程专业，但当前投入最多的是软件开发与产品落地。正在持续推进的核心项目是 HarmonyOS 应用看板 App，目前已进入邀请测试阶段，并规划后续 macOS 与 Android 版本。

除应用开发外，我也在做独立的 OpenHarmony Hi3861 嵌入式开发实践，这条线与应用看板分开推进，重点在设备侧能力、系统理解与工程实现。

我长期参与开源协作：在 Sweet Potato Mod 中与 @teddyxlandlee 协作，在 Difficlut Rocket 项目中与 @shenjackyuanjie 协作；我的 Minecraft Java ID 是 Rayawa。

学术与跨学科方面，我完成并整理过 PCR 相关项目、蛋白质研究设想、碳同化综述，以及橙色平板实验记录，并把一部分实验内容做成了可访问的网页化项目。

网页开发上，我做过个人网站、模组网站、第五人格相关站点、农杆菌转化法实验站点等；同时持续学习 Java 与 Rust，并计划推进仓颉开发，最近也在学习法语。

生活与兴趣上，我喜欢跑步和骑行，常拍风景摄影并制作无人机视频；游戏方面长期接触 Minecraft、原神、明日方舟、Phigros、Arcaea、舞萌、Lanota、Rizline、皇室战争与第五人格，也把这些社群经验反哺到创作表达中。
