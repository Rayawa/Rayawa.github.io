Sweet Potato Mod

农业与魔法的完美结合！与@teddyxlandlee协作开发的Minecraft模组项目，添加了地瓜、农林台、研磨器、魔法锅、附魔作物等。

Pigeonia Featurehouse 主页 GitHub 下载链接

新的内容

地瓜

在本模组中添加的类食物。它分为紫薯、红薯和白薯，以及三个变种，有地瓜粉、地瓜皮等。

魔法锅

魔法锅散发着黑色的光泽。你可以通过在它下方3×3区域内点燃灵魂火来激活魔法锅，将三个任意品类的生地瓜转换为其他品类或对其进行随机附魔。

研磨器

研磨器有着一个磨盘，它的作用是将地瓜研磨为地瓜粉。

附魔树苗和作物

附魔树苗和作物的生长速度要快于原版树苗或作物，且会长成附魔树或附魔作物。

注意事项

  • 目前该模组仅支持 Fabric 1.16+，更早版本的玩家暂时不支持。
  • 请不要在未经作者允许的情况下擅自转载本模组至其它平台！
  • 加载本 Mod 需要 Fabric Loader 和 Fabric API。其中 Loader 的版本不能低于 0.9.0，API 最好为最新版。
  • 不要在未经本模组作者允许的情况下擅自把 Mod 中的内容添加到你的整合包中！
  • 如果您想把本模组添加至你的服务器中，请同时在客户端与服务端安装该 Mod。
  • 如果您发现了模组中的 Bug 或对本模组有建议，烦请汇报至 GitHub Issues。
