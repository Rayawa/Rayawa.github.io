修复个人简介块和板块大小上限问题

❌ 已发现的问题：

  1. 个人简介块在移动端没有纵向排布 - 因为桌面端的 display: flex 在移动端没有被重置
  2. 板块大小上限被限制 - .site-shell 的最大宽度限制太小
  3. 大屏幕下板块不会跟着变大 - 各种最大宽度限制阻碍了响应式扩展

测试方法：缩小浏览器窗口到移动端尺寸（＜768px），查看上方三个模块是否变成单列纵向排列。

✅ 已实施的修复：

  1. 个人简介块移动端修复 - 在移动端媒体查询中添加 .intro-module { display: block; }
  2. 提高 .site-shell 最大宽度 - 从 1320px 提高到 1600px
  3. 提高 1200px 断点最大宽度 - 从 1100px 提高到 1400px
  4. 提高平板端最大宽度 - 从 760px 提高到 900px
  5. 移除模块最大宽度限制 - 添加 .module { max-width: none; }
  6. 提高项目卡片最小宽度 - 从 240px 提高到 280px
  7. 提高链接卡片最小宽度 - 从 220px 提高到 260px
